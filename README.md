## Diseño del robot
Para realizar el diseño del robot se utilizo la siguiente configuraciòn dentro del model editor de gazebo

Chasis:

[X, Y, Z] = [0.3, 1.0, 0.1]

Llantas:

    • Z pose = 0.2
    • Roll = 1.5707 rad
    • Visual and Collision geometry: [Radius, Length] = [0.2, 0.1]
